.files
======
Personal macOS configuration and _[Homebrew] [tap]_.

* [Formula]e
* [_Cask_]s
* _[Brewfile]_ including _[mas]_<sup>*</sup> apps

\* [Mac App Store]

Install
-------
~~~ sh
brew tap danielbayley/dotfiles
brew install #formula
brew cask install #cask
~~~
`git config core.hooksPath hooks`
