_[Homebrew]_**ery**
===================
Personal [Homebrew] setup.
* [Formulae]
* [_Cask_]s
* _[Brewfile]_ including _[mas]_<sup>*</sup> apps

\* [Mac App Store]

Install
-------
~~~ sh
brew tap danielbayley/homebrew-ery
brew install #formula
brew cask install #cask
~~~
